# React Contact Book

This project is a simple contact book application built with React and Vite. It allows users to add, search, and manage contacts efficiently.

## Features
- Add new contacts with name and phone number
- Search contacts by name
- View a list of all contacts
- Delete contacts
- Responsive and modern UI

## Technologies Used
- React
- Vite
- CSS Modules

## Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ozncncyr/goit-react-hw-03.git
   ```
2. Navigate to the project directory:
   ```bash
   cd goit-react-hw-03
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the App
Start the development server:
```bash
npm run dev
# or
yarn dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.
